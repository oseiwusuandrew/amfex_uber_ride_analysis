# Data Directory

This directory contains the datasets used in the Uber ride analysis project.

## Files

### Uber_Data.xlsx
Original dataset containing 297 Uber ride records from 2024.

Columns:
- RideID: Unique identifier for each ride
- Date: Date and time of the ride
- PickupLocation: Starting location
- DropoffLocation: Destination location
- Distance (km): Distance traveled in kilometers
- FareAmount ($): Fare charged in dollars
- TimeOfDay: Period of day (Morning, Afternoon, Evening, Night)
- PaymentMethod: Payment type (Wallet, Card, Cash, Mobile Money)

### Uber_Data_Cleaned.csv
Cleaned and processed dataset generated by the analysis script.

Additional columns:
- FarePerKM: Calculated fare per kilometer
- Month: Extracted month name from date
- DayOfWeek: Extracted day of week from date
- YearMonth: Year-month period for trend analysis
- IsWeekend: Boolean flag for weekend rides

## Data Quality

- No missing values
- No duplicate records
- All distance and fare values validated as positive
- Date format standardized to datetime
- 100% data completeness

## Usage

The original dataset (Uber_Data.xlsx) is used as input for the analysis script.
The cleaned dataset (Uber_Data_Cleaned.csv) is generated automatically when running the analysis.
